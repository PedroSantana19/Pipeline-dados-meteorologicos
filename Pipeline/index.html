Extração de dados metereológicos

Objetivo do Projeto

Fomos contratados por uma agência de turismo de Boston que organiza excursões para turistas vindos de diversos países. Para definir os melhores roteiros para cada dia, é importante contar com dados atualizados de previsão do tempo. Nosso trabalho será extrair os dados de uma API e disponibizar um arquivo contendo esses dados de previsões dos próximos 07 dias em uma pasta definida pelo time de negócio.

Tecnologias e libs utilizadas

  • Python;
  • Pendulum;
  • DAG;
  • Airflow;
  • API REST;
